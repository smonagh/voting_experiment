The procedure on your computer screen

The screens in which you make your decisions are structured as follows.

1) The first screen in each period informs you about the number of votes each Type A participant holds and the number of votes that are required to recommend a project to the Type B participant. From this screen, you will learn your subject number and, importantly, how many votes you control in this period.

You are one of the participants of type A. Specifically, you are now participant A1. You have two votes, the sum of the votes is 7, and 4 votes are needed for a decision.

2) In each period, the second screen that you and the other Type A participants see , informs you about the value of Project X and the value of Project Y. All participants of Type A are informed about the values of both projects at the same time. The Type B participant has no information about the value of either of the two projects. The Type B participant is also not informed about the procedure (total number of votes, votes assigned to each voter, the 50% majority of votes required for a message to be sent to Type B) by which the group of Type A participants decides on the message to be sent.

In the example above, if Participant B chooses Project X, Participant B receives E$8. And if Participant B chooses Project Y, Participant B receives E$12.

3) In each period, the third screen is the decision screen. Here, you will vote on whether Message X or Message Y should be sent to the Type B participant. On this decision screen, you can always see in the upper row whether you are subject A1, A2, or A3.

Note, the bottom of this screen tells you how many votes are needed to make a decision as to whether to send Message X or Message Y.

Using the computer mouse, please click on the field that indicates the decision that you want to make.

After you have clicked the button for Message X or Message Y, click on the next button to move to the next period, where you may be allocated a new subject number. With this new subject number you might also have a different number of votes, because each subject number is tied to the number of votes you see in the previous screen. There will be five periods of this part (Part 1) of the experiment, so that you are once A1, once A2, etc. in a random order, until you have cycled through all the five possibilities.

4) Part 1 is ended by the following screen.

 
   

Part 1 is complete. Please click next and continue to the wait screen while the other players finish making their decisions. Part 2 will start when everyone has finished.

   
 

Once you have seen this screen, please press Next. You will then see a screen that announces the start of Part 2.

Part 2 proceeds exactly as Part 1, except that some of the A1, A2, etc. have a different numbers of votes than in Part 1.

After the second part is completed you will see a screen that summarizes your actions for both Part 1 and Part 2 of the experiment and your personal earnings across all 6 periods.
